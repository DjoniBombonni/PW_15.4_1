# GALAGA - игра на языке C

Классическая аркадная игра Галага, реализованная на языке C с использованием ASCII символов.

## Особенности

- Управление персонажем через клавиатуру (A/D для движения, Space для выстрела)
- Враги нападают сверху экрана
- Система очков и волн
- Система здоровья игрока
- ASCII графика и рамки интерфейса

## Управление

- **A** - движение влево
- **D** - движение вправо  
- **Space** - выстрел
- **Q** - выход из игры

## Требования

- Linux/macOS/Unix
- GCC компилятор
- ncurses библиотека (libncurses-dev)
- Терминал размером минимум 80x24

## Установка ncurses

**Ubuntu/Debian:**
```bash
sudo apt-get install libncurses-dev
```

**macOS:**
```bash
brew install ncurses
```

## Компиляция

```bash
gcc -o galaga main.c -lncurses
```

## Запуск

```bash
./galaga
```

## Правила игры

1. Управляй кораблем (^) снизу экрана
2. Уничтожай врагов (V) перед тем как они достигнут тебя
3. Выстрелы (|) движутся вверх
4. За каждого уничтоженного врага получаешь 10 очков
5. Каждая новая волна враги спускаются чаще и быстрее
6. Игра заканчивается когда здоровье достигает нуля

## Система сложности

Сложность растет с каждой волной:
- Враги появляются чаще
- Нужно уничтожить всех врагов чтобы перейти на следующую волну
